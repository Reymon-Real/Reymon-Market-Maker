# Anwendung der Formel

Den fairen Preis anhand der Formel `Mikropreis = M + G` schätzen

# Aufschlüsselung der Grundformel

$M = (bid_price + ask_price) / 2$
$S = ask_price − bid_price$
$I = bid_size / (bid_size + bid_ask)$
$G = (I − 0.5) × S$

$Mikropreis = M + G$
